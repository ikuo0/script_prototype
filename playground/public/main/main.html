スクリプト言語の概要

基本構文

MAIN / END_MAIN

スクリプトのエントリーポイントを定義する。

コメントは ; を使用する

関数や引数の区切り文字は半角空白


            MAIN; メイン処理
              PRINT "Hello, World!"
            END_MAIN

変数の定義、代入

変数は DATA で定義し、 DATA で代入する。


            MAIN
              DATA a 1; 変数 a を定義し、1 を代入
              DATA b 2; 変数 b を定義し、2 を代入
              DATA a 9; 既存の変数 a に 9 を代入
            END_MAIN

関数の定義　FUNCTION / END_FUNCTION / CALL_FUNCTION

変数は全て参照渡しであるため関数内の変更は呼び出し元にも反映される

途中で処理を中断する場合は RETURN する

定義した関数は CALL_FUNCTION で呼び出す


            FUNCTION SUM3 a b c result
              MATH_ADD result a b
              MATH_ADD result result c
              RETURN
            END_FUNCTION

            MAIN
              DATA a 1
              DATA b 2
              DATA c 3
              DATA result 0
              CALL_FUNCTION SUM3 a b c result
              PRINT "@result"
            END_MAIN

制御構文

条件分岐（IF / ELSE）

条件分岐は IF / ELSE で行う

IF文は変数を１つ指定し True 分岐する、NOTや計算式は判定できない


            MAIN
              DATA a 1
              DATA b 2
              DATA flag 0
              CMP_GREATER flag a b; a > b なら flag に 1 が代入される
              IF flag
                PRINT "a is not greater than b"
              ELSE
                PRINT "a is greater than b"
              END_IF
            END_MAIN

繰り返し（LOOP / END_LOOP）

指定した回数処理を繰り返す、1 開始であるため LOOP 3 i とした場合には 1, 2, 3 とカウンタがインクリメントされる


            MAIN
              LOOP 3 i
                PRINT "i=" "@i"
                LOOP 4 j
                  PRINT "j=" "@j"
                END_LOOP
              END_LOOP
            END_MAIN

組み込み関数

出力（PRINT）

標準出力に文字列を出力する

ダブルクォーテーションで囲った文字を出力可能

ダブルクォーテーションで囲い @ を先頭に付けた変数を出力可能


            MAIN
              PRINT "Hello World!"; 通常の文字列出力
              DATA a 1
              DATA b 2
              DATA flag 0
              CMP_LESS flag a b; a < b なら flag に 1 が代入される
              PRINT "flag=" "@flag"; 変数の出力
            END_MAIN

命令一覧

MAIN / END_MAIN

メイン処理の開始と終了を示すブロック構文です。この範囲内に実行されるコードを記述します。

FUNCTION / END_FUNCTION / RETURN

関数の定義を行うための構文です。FUNCTION で開始し、END_FUNCTION で終了します。RETURN によって関数の戻り値を指定できます。

DATA

変数の新規作成および初期値の設定を行います。例：DATA @x, 100

IF / ELSE / END_IF

条件分岐を行う構文です。IF で条件を判定し、必要に応じて ELSE で分岐し、END_IF で終了します。

LOOP / END_LOOP

ループ処理の開始と終了を示すブロックです。LOOP で繰り返しを開始し、END_LOOP まで繰り返します。

CALL_FUNCTION

定義済みの関数を呼び出します。引数および戻り値は参照によってやり取りされます。

MATH_ADD

加算を行う命令です。形式：MATH_ADD result lhs rhs（result に lhs + rhs の結果を格納）

MATH_SUB

減算を行う命令です。形式：MATH_SUB result lhs rhs（result に lhs - rhs の結果を格納）

MATH_MUL

乗算を行う命令です。形式：MATH_MUL result lhs rhs（result に lhs × rhs の結果を格納）

MATH_DIV

除算を行う命令です。形式：MATH_DIV result lhs rhs（result に lhs ÷ rhs の結果を格納）

CMP_GREATER

比較命令。lhs が rhs より大きい場合に true を result に格納します。

CMP_GREATER_EQUAL

lhs が rhs 以上であるかを比較し、結果を result に格納します。

CMP_LESS

lhs が rhs より小さいかを比較し、結果を result に格納します。

CMP_LESS_EQUAL

lhs が rhs 以下であるかを比較し、結果を result に格納します。

CMP_EQUAL

lhs と rhs が等しいかを比較し、結果を result に格納します。

CMP_NOT_EQUAL

lhs と rhs が異なるかを比較し、結果を result に格納します。

PRINT

指定された値や変数の内容を標準出力に表示します。例：PRINT @message
